# 类型判断相关函数

## isObject
### 参数
- 接收一个参数 `val`，返回值类型为 `boolean`

| **参数属性** | **说明**     | **类型** |
| ------------ | ------------ | -------- |
| `val`        | 需要判断的值 | `any`    |

## isDate
### 参数
- 接收一个参数 `val`，返回值类型为 `boolean`

| **参数属性** | **说明**     | **类型**  |
| ------------ | ------------ | --------- |
| `val`        | 需要判断的值 | `unknown` |
